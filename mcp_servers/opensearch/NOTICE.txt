本项目的新增/修改(Modifications by AWS DevOps MCP Platform)
-----------------------------------------------------------
为适配 AWS Bedrock AgentCore Runtime 容器契约,本项目新增了以下文件
(不属于 upstream,为本项目原创的适配层):

  entrypoint.py        — 容器引导层:
                           1) 从 AWS Secrets Manager 拉取 basic auth 凭据注入环境变量
                           2) force-basic-auth:中和调用方注入的 aws_iam_arn/profile/
                              serverless 认证参数,锁死走 basic auth
  Dockerfile           — linux/arm64 容器,监听 0.0.0.0:8000 /mcp(AgentCore 契约)
  requirements.txt     — 依赖清单(版本沿用 upstream pyproject.toml)
  .dockerignore        — 构建忽略清单
  README.md            — 本项目使用文档

upstream 源码(mcp_server_opensearch/ opensearch/ tools/)**保持不改**;
所有本项目特有的适配逻辑均外挂在上述新增文件中。
